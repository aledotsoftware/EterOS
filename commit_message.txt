Fix cmd_net network status padding alignment and offset

This commit updates the padding inside `cmd_net` to output a fixed number of lines regardless of the state of the network adapter or DHCP status. This fixes UI alignment issues inside `cmd_panel.c` where the hitboxes map to fixed Y-coordinates.
Additionally, it updates `cmd_panel.c` so that the UI options correctly match the fixed 7 lines of output generated by `cmd_net.c`.
